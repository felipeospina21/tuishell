# tuishell Roadmap

## Should Have

- [ ] `CHANGELOG.md` — track breaking changes between tagged releases
- [ ] Unit tests for pure functions (`layout_test.go`, `utils_test.go`)
- [ ] `example/go.mod` — standalone module so the example is runnable independently and visible on pkg.go.dev
- [ ] GoDoc comments on all exported types, functions, and methods

## Nice to Have

- [ ] `CONTRIBUTING.md` — contribution guidelines for external contributors
- [ ] CI workflow (`.github/workflows/ci.yml`) — `go vet`, `go test`, `golangci-lint` on push
- [ ] Tagged releases with semantic versioning (`v0.1.0`, etc.)
